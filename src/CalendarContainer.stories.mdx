# CalendarContainer

The `CalendarContainer` component is used internally based on `dateFormat`, `timeFormat` and various internal `viewMode`, `selectedDate`, `viewDate` states.

## Days

```jsx
<CalendarContainer viewMode="days" />
```

<Preview>
  <Story name="Days viewMode">
    <CalendarContainer viewMode="days" />
  </Story>
</Preview>

The `selectedDate` is highlighted.

```jsx
<CalendarContainer viewMode="days" selectedDate={new Date()} />
```
