# CalendarContainer

The `CalendarContainer` component is used internally based on `dateFormat`, `timeFormat` and various internal `viewMode`, `selectedDate`, `viewDate` states.

## Days

```jsx
<CalendarContainer viewMode="days" />
```

<Preview>
  <Story name="Days viewMode">
    <CalendarContainer viewMode="days" />
  </Story>
</Preview>

The `selectedDate` is highlighted.

```jsx
<CalendarContainer viewMode="days" selectedDate={new Date()} />
```

<Preview>
  <Story name="Days selectedDate">
    <CalendarContainer viewMode="days" selectedDate={new Date()} />
  </Story>
</Preview>

The picker can be formatted via the `locale` prop on `DateTime`, which is forwarded to `CalendarContainer` via `formatOptions`.

```jsx
<CalendarContainer viewMode="days" formatOptions={{ locale: es }} />
```

<Preview>
  <Story name="Days locale">
    <CalendarContainer viewMode="days" formatOptions={{ locale: es }} />
  </Story>
</Preview>

## Time

```jsx
<CalendarContainer viewMode="time" />
```

<Preview>
  <Story name="Time viewMode">
    <CalendarContainer viewMode="time" />
  </Story>
</Preview>

Format to military time via `timeFormat`.

```jsx
<CalendarContainer
  viewMode="time"
  timeFormat={`${FORMATS.MILITARY_HOUR}:${FORMATS.MINUTE}`}
/>
```

<Preview>
  <Story name="Time military time">
    <CalendarContainer
      viewMode="time"
      timeFormat={`${FORMATS.MILITARY_HOUR}:${FORMATS.MINUTE}`}
    />
  </Story>
</Preview>

Customize time precision via `timeFormat`.

```jsx
<CalendarContainer
  viewMode="time"
  timeFormat={`${FORMATS.HOUR}:${FORMATS.MINUTE}:${FORMATS.SECOND}.${FORMATS.MILLISECOND}`}
/>
```

<Preview>
  <Story name="Time with precision">
    <CalendarContainer
      viewMode="time"
      timeFormat={`${FORMATS.HOUR}:${FORMATS.MINUTE}:${FORMATS.SECOND}.${FORMATS.MILLISECOND}`}
    />
  </Story>
</Preview>

When dates are supported, show a switcher via `dateFormat`.

```jsx
<CalendarContainer
  viewMode="time"
  dateFormat={`${FORMATS.MONTH}/${FORMATS.DAY}/${FORMATS.YEAR}`}
/>
```

<Preview>
  <Story name="Time dateFormat">

      dateFormat={`${FORMATS.MONTH}/${FORMATS.DAY}/${FORMATS.YEAR}`}
    />
  </Story>
</Preview>
